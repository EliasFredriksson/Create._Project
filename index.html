Hi and welcome to Create!

This website is a practise project about the development process of websites design. I started with a graphical profile. Then wireframes and mockups. And finally this website. It was really fun to learn alot about the Adobe programs. Specifically Illustrator, Photoshop and InDesign.

Check out the graphical profile, wireframes and mockups in the About tab.

The idea of Create. was to create a company who provide a documentation service to other programming languages. Or anything that needs documentation. As I feel there is a need for more clear, easy to read and understand documentation out on the internet.

Everything you see here on this website is made with gradients and vector graphics that have been exported from Adobe Illustrator into svg files.
